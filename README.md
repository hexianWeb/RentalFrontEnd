# 民宿租赁平台用户端 微信小程序

> 作者：何贤

## 前言

整个项目包含27个页面，涉及实时聊天、页面信息展示、地图展示导航等。写这个项目主要是练习和熟悉微信小程序和如何在微信小程序里使用websocket

## 技术栈

----------

```
微信小程序
```



## 项目运行

````
git clone https://github.com/hexianWeb/RentalFrontEnd.git
cd miniweixin
npm install 
````
